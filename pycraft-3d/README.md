# PyCraft 3D

## Overview
PyCraft 3D is a Minecraft-style game developed using Pygame. The game features a 3D world where players can explore, build, and interact with various elements. It supports keyboard and mouse inputs for movement and camera control.

## Features
- 3D world rendering
- Player movement and interaction
- Chunk-based world generation
- Customizable blocks and items
- User interface for game controls and settings

## Installation
To get started with PyCraft 3D, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd pycraft-3d
   ```

2. **Install dependencies:**
   Make sure you have Python and Pygame installed. You can install the required packages using:
   ```
   pip install -r requirements.txt
   ```

3. **Run the game:**
   Start the game by executing the main script:
   ```
   python src/main.py
   ```

## Directory Structure
- `src/`: Contains the source code for the game.
  - `main.py`: Entry point of the application.
  - `pycraft/`: Package containing game logic and components.
  - `assets/`: Directory for models, shaders, and sounds.
  - `tests/`: Contains unit tests for the game functionalities.
  
- `requirements.txt`: Lists the dependencies required for the project.

- `pyproject.toml`: Project configuration file.

- `.gitignore`: Specifies files and directories to be ignored by Git.
